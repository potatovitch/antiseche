# olaleTCPdump

## Capture de trames

### tcpdump

> tcpdump : dump des trames

```bash
tcpdump -D
tcpdump -c <nbpacket>
tcpdump -i <interface/ip jsp flm>
```

#### OPTIONS

- -D : list interface
- -c : count le nbpacket puis quitte quand il a atteint le count
- -i : donne les trammes de l'interface choisit et du type choisit
